# How to Run (Simple Version)

Requires:
1. **Python 3.10+** (Download from [python.org](https://www.python.org/downloads/))
2. **Node.js 18+** (Download from [nodejs.org](https://nodejs.org/))
3. **FFmpeg** (See instructions below if you don't have it)

## Mac / Linux Users
1. Open a terminal.
2. Drag `start_app.sh` into the terminal and press Enter (or run `./start_app.sh`).
3. The app will open in your browser automatically.

## Windows Users
1. Double-click `start_app.bat`.
2. The app will open in your browser automatically.

## Common Issues
- **"FFmpeg not found"**: 
  - **Mac**: Install Homebrew, then run `brew install ffmpeg`.
  - **Windows**: Download FFmpeg build, extract it, and add the `bin` folder to your PATH (or Google "how to install ffmpeg on windows").
